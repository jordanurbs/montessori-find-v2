Static Asset Test

If you can see this box with a green background, static assets are being served correctly.

This page is a simple test to confirm if Netlify is serving static assets correctly.

If this page displays properly but your main site doesn't have CSS, the issue is likely with how Next.js is processing Tailwind CSS during the build process.

Debugging Steps:

  1. Check browser console for any CSS loading errors
  2. Verify that the build process is including CSS files
  3. Check that the Next.js plugin for Netlify is correctly configured
